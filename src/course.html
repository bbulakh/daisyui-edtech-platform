Course Description

This HTML core concepts course covers the fundamental principles and elements of HTML (Hypertext Markup Language), a language used to create and structure content on the web. Students will learn how to use HTML tags to create headings, paragraphs, links, images, and lists. They will also learn about HTML syntax, the document structure, and the different versions of HTML. By the end of the course, students will have a solid understanding of HTML and the ability to create basic web pages. No prior coding experience is required.

Main topics

  • Introduction to HTML
  • HTML Tags
  • HTML Elements
  • HTML Attributes
  • HTML Forms
  • HTML Links
